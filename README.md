# Tms-watch-list-adder

It adds company name to watch list in tms

## To run the project
1. Clone the repository
   ```bash 
   git clone https://github.com/aryalsuman/Tms-watch-list-adder.git
   ``` 
2. Run tmsadder.py file
    ```bash 
    python tmsadder.py
     ```
